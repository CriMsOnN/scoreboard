About Scoreboard

sb

What is it?

A simple app that lets you create a realtime scoreboard for whatever you want. No rules.

How was it built?

Backend: Node.js, Socket.io, and Mongoose.js

Frontend: Backbone.js

It's a work in progress! I intentionally made it very plain because I didn't want to get caught up in the design until the functionality is complete.

What's next

User Account/Game Ownership: Right now anyone can update a score, not cool.

Performance: Currently when you update the score, there is some lag because it only broadcasts the score change after Mongodb has been updated. This is fine for viewers, but makes it seem chokey for the scorekeeper.

Design: I gotta make it pretty

Test: Find out what sucks and fix it
